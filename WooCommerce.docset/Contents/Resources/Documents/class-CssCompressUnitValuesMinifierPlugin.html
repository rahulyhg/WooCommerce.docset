Class CssCompressUnitValuesMinifierPlugin

This aCssMinifierPlugin will compress several unit values to their short notations. Examples:

padding: 0.5em;
border: 0px;
margin: 0 0 0 0;

Will get compressed to:

padding:.5px;
border:0;
margin:0;
