Class CssVariablesMinifierPlugin

This aCssMinifierPlugin will process var-statement and sets the declaration value to the variable value.

This plugin only apply the variable values. The variable values itself will get parsed by the CssVariablesMinifierFilter.

Example: <code>

aCssMinifierPlugin
